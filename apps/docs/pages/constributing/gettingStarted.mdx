# Getting Started

There you can find introduction to the project, how project is build,works and more.

    ## Setup Project
    <Callout type="warning" emoji="⚠️">
        PNPM is required to install dependencies. If you don't have it, install it with `npm i -g pnpm`.
    </Callout>

    <Steps>
        ### Download the project

        ```bash copy
            git clone https://github.com/ineedjet/fluidui.git ./fluidui
        ```

        ### Install Dependencies

        ```bash copy
            cd fluidui
            pnpm i
        ```
        ### Run Project!
        Right now you can run project by using `pnpm dev` command.
        (For more scripts check [Enviroment](/layers/enviroment)))

        ```bash copy
            pnpm dev
        ```
    </Steps>

    ## Implementation Process
    <Steps>
        ### Write Documentation
        On start we are creating documentation to make sure that we know what we want to do.

        ### Write Tests
        TDD is a good way to write code, so we use it to write tests before code.

        ### Write Code
        Write code that pass tests.

        ### Improve Documentation
        After writing code we are improving documentation to make sure that it's up to date.

        ### ChangeLogs
        Use ChangeSet to create changelogs for your changes.

        ### Commit
        Commit your changes.

    </Steps>
