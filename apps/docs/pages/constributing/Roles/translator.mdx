# Translator

Translators have important role in our project. With them we can correctly apply translates at

<abbr title="Internationalization">I18n</abbr> into our project.

    ## About Constributing

    ### Example of Translation needed
    - Socials
        - Youtube
            - Subtitles for videos (YouTube)
    - Docs
        - Translate pages
    - Package
        - Translate mostly single words at `.json` files.

    ### Targeted Languages
    At the moment we are targeting these languages:

    - English (US)
    - Polish
    - Spanish
    - French
    - German
    - Italian
    - Russian
    - Chinese
    - Turkish
    - Japanese
    - Hindi


    ### Language Codes
    We use [ISO 639-2](https://www.loc.gov/standards/iso639-2/php/English_list.php) for 2 char language codes (usually at json files).
